# Rent Calculator (Python)

## Overview
A simple Python script to split rent, food, and electricity costs among roommates.  
This project demonstrates clean modular code, input validation, and practical problem-solving.

## Features
- Input: rent, food expenses, electricity units, charge per unit, number of persons
- Output: per-person share of total expenses
- Input validation to avoid crashes
- Modular functions (`calculate_share`, `get_int_input`, `main`) for clarity and reusability

## Usage
Run the script in your terminal:

```bash
python rent_calculator.py
